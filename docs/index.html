Umwandlung zur Normalform

Nur wenn a \neq 0

\begin{align*} ax^2 + bx + c &= 0 \\ x^2 + \frac{b}{a}x + \frac{c}{a} &= 0 \\ x^2 + px + q &= 0 \\ \end{align*} \implies p = \frac{b}{a}, \; q = \frac{c}{a}
Umwandlung zur Normalform

Ist schon normal!

x^2 + px + q = 0
Umwandlung zur Normalform

Nur wenn a \neq 0 \begin{align*} a \cdot (x - x_1) \cdot (x - x_2) &= 0 \\ (x - x_1) \cdot (x - x_2) &= 0 \\ x^2 - x_2 x - x_1 x + x_1 x_2 &= 0 \\ x^2 - \left( x_1 + x_2 \right) x + x_1 x_2 &= 0 \\ x^2 + px + q &= 0 \end{align*}

\implies p = - \left( x_1 + x_2 \right), \; q = x_1 x_2

Das ist Vietas Satz!

Umwandlung zur Allgemeinform

Ist schon allgemein!

ax^2 + bx + c = 0
Umwandlung zur Allgemeinform

Nur wenn a \neq 0

\begin{align*} x^2 + px + q &= 0 \\ x^2 + \frac{b}{a}x + \frac{c}{a} &= 0 \\ ax^2 + bx + c &= 0 \end{align*}
Umwandlung zur Allgemeinform

Nur wenn a \neq 0 \begin{align*} a \cdot (x - x_1) \cdot (x - x_2) &= 0 \\ a \cdot \left[ (x - x_1) \cdot (x - x_2) \right] &= 0 \\ a \cdot \left[ x^2 - x_2 x - x_1 x + x_1 x_2 \right] &= 0 \\ a \cdot \left[ x^2 - \left( x_1 + x_2 \right) x + x_1 x_2 \right] &= 0 \\ a \cdot \left( x^2 + px + q \right) &= 0 \\ a \cdot \left( x^2 + \frac{b}{a}x + \frac{c}{a} \right) &= 0 \\ ax^2 + bx + c &= 0 \end{align*}

Beziehungen und Umwandlungen
